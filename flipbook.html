Sekatan Keselamatan Browser (Ralat CORS)

Sistem mengesan pelayar web anda (Chrome/Edge/Safari) menyekat pemuatan fail PDF tempatan atas faktor keselamatan.

Pilihan Penyelesaian Mudah:

1. Guna Mozilla Firefox: Firefox membenarkan fail PDF tempatan dibuka terus secara offline tanpa sebarang ralat.

2. Guna Local Server: Jalankan laman web ini menggunakan sambungan pelayan tempatan (seperti VS Code Live Server).
